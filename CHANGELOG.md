# Changelog

## [0.1.0 Alpha] - 2026-02-01
### Added
- Initial project structure for Forge 1.20.1.
- **Custom Main Menu**: Modern dark UI with direct access buttons.
- **KillAura Module**: Automatically attacks nearby entities.
  - Customizable Range, Target selection (Players/Mobs), and Rotation.
- **Telekinesis Module**: Pick up and throw entities using right-click.
- **Localization**: Full support for English (en_us) and Turkish (tr_tr).
